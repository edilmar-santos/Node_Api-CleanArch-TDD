# CleanArch - App

Api Rest utilizando conceitos de Clean Architecture, Design Patterns e TDD.

# Tecnologias e pacotes utilizados
  * Node.js
  * Express
  * mongodb
  * jest
  * husky
  * lintstaged
  * bcrypt
  * jsonwebtoken
  * validator


# Executar testes unitários
  npm run test:unit

# Executar testes integrados
  npm run test:integration

# Executar aplicação
  npm run start
